Макрос для вставки номеров рисунков, v. 1.0
Автор -- Сергей Хасанов <s.a.khasanov@gmail.com>

Установка:
скопируйте файл Figurator.gms в папку GMS (обычно располагается в C:\Program Files\Corel\CorelDRAW Graphics Suite X5\Draw\)

Запуск:
Инструменты->Макросы->Запустить макрос, выберите Figurator.NewFigureNumbering из FigureNumbering (Figurator.gms)
Нажмите Run.

Enjoy :)
